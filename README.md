# gawl
**GL** + ** **Wa**yland  
Tiny C++ library for OpenGL on Wayland  
Written with templates in order to reduce overheads

# Dependencies
[towl](https://github.com/mojyack/towl)  
FreeType 2.10.4~  
ImageMagick 7.0~  

# Build
Meson and Ninja required
```
meson build && ninja -C build
```
